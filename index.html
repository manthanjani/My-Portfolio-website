In the academic year 2020 I had made a project named “Face Mask Recognition”. It was basically an ML-based real-time machine learning model using a Convolutional neural network and MobileNetV2 which detects masks. It functions as we run the program, a camera will be opened and it will detect a face. If the person is wearing a mask, then a green box will appear on the face and accuracy will be visible at the bottom of that box. And if the person is not wearing a mask a red box will appear on his face. The accuracy of the Model is approximately 98%.
